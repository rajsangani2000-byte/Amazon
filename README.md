## 📌 Project Description

This project is a fully responsive **Amazon-inspired e-commerce website** built to replicate the core design and functionality of the original Amazon platform.

It includes product listings, shopping cart functionality, user-friendly UI, and a modern responsive layout.

---

## 🚀 Features

* 🏠 Homepage with hero section
* 🛍️ Product listing page
* 🔍 Product search functionality
* 🛒 Add to cart feature
* 💳 Checkout page UI
* 📱 Fully responsive design
* 🎨 Clean and modern UI inspired by Amazon
